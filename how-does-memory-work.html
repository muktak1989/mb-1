How does memory work?

When you remember something, you don’t recall the event itself but the meaning behind it. What you remember depends on the context in which the event occurred. For example, if you saw a red car in a blue neighborhood, you would remember that red is a color associated with danger and blue is a color associated with calmness. Similarly, if you see a hammer, you would remember that it’s a tool used in construction. These associations are known as “concrete” memories because they are specific and direct. They are also fairly fleeting since they are linked to the event and not necessarily the event itself. Other types of memories, known as “abstract” memories, are not directly tied to a specific event but instead are general ideas or thoughts related to a certain concept.
